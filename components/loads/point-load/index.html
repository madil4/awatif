Point Load Template

A point load template that allows users to apply concentrated forces and moments at geometry points.

Parameters

  • Fx: Force component in X direction
  • Fy: Force component in Y direction
  • Fz: Force component in Z direction
  • Mx: Moment component about X axis
  • My: Moment component about Y axis
  • Mz: Moment component about Z axis

Usage

The point load is applied to geometry points. When a geometry point maps to multiple mesh nodes (via geometryMapping), the load is applied to all corresponding nodes.

Example


import { pointLoad } from "./pointLoad";

// Create a point load with downward force of 1000N
const load = pointLoad.getLoad({
  params: {
    Fx: 0,
    Fy: 0,
    Fz: -1000,
    Mx: 0,
    My: 0,
    Mz: 0,
  }
});

// Result: { load: [0, 0, -1000, 0, 0, 0] }
